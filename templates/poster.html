1-VALID Add basic data validation

To validate data I edited the app.py file. All the data processing SHOULD ocurr in the 'GET' script as that is when the form is returned to us with the data. Within this script we must first load all the data inputs from the form using var = request.form['var']. After loding all the variables in I used if and elif loops to return error messages. For basic data such as numbers I used the .isdigit() syntax to find out if the input was a number, if it wasnt then i returned a error.

To pass the error back to user I added up the errors to a string variable msg which I then returned with the buggy-form.html template. I used {{msg}} which allowed me to display the error messages in the returned buggy form template.
