0-GET Get the source code

I forked the repo onto my GitHub and then used the $ git clone ssh://github.com/mahid2000/CS1999-buggy-race-editor.git command on the git bash terminal to clone the repository onto my local machine.

Once the repo was on my own machine I was able to commit changes to version control as I went along, and I pushed back up to my GitHub repo at the end of each day. I followed GitHub's instructions and set up a SSH key so I didn't need to enter username and password every time I pushed.

0-RUN Get app running and view it in a browser

I opened a git bash to the cloned buggy editor directory on my local machine and installed all the required python libraries using pip install -r requirements.txt . To run the webserver I used the command Python app.py which output * Running on http://0.0.0.0:5000/ telling me the adress of the webpage which I copied and pasted into my browser to be greeted with the index page of the buggy editor.

I used ssh instead of https as I had already set up the ssh key on my local machine allowing me easy secure access to github without having to log-in evry time.
